# Launch kit: The Real-World Email Regex

Promotion assets for https://emailregex.honestcode.software/. These files are not deployed (they
live outside `public/`). Work the list top to bottom; it is ordered by effort-adjusted ROI.

## Recommended sequence

1. **HackerNoon relink** (`1-hackernoon-relink.md`) — 30 seconds, dofollow backlink. Do this first.
2. **Show HN** (`2-show-hn.md`) — Tue to Thu, ~8-10am ET. Submit URL, post the first comment instantly,
   then stay in the thread for the first 2 hours.
3. **Reddit** (`3-reddit.md`) — r/programming and r/webdev on different days. Also feeds AI search.
4. **dev.to republish** (`4-devto-republish.md`) — has `canonical_url` set, so it builds your site's
   authority instead of competing. Cross-post to Hashnode and Medium with the same canonical if you
   want more reach.
5. **YouTube** (`5-youtube.md`) — the compounding bet. Script, title, thumbnail, description with
   chapters, and a shot-by-shot storyboard are all ready; you provide the recording.

## Timing

Do not fire everything the same day. A good cadence:

- Day 1: HackerNoon relink + Show HN.
- Day 2-3: r/programming, then r/webdev.
- Day 4-5: dev.to (and mirrors).
- Week 2+: YouTube, once you have a recording.
